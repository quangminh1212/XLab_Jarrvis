# Jarvis Voice Call

Real-time direct conversation with AI like a phone call.

Features:
- Continuous mic listening (voice activity detection)
- Real-time STT (Vietnamese)
- AI response (placeholder for now)
- Instant TTS voice reply
- Phone-call style: speak → AI listens → replies → repeat

Run:
python jarvis_call.py

Requirements:
- sounddevice
- speech_recognition
- numpy
- edge-tts or pyttsx3 (for TTS)
